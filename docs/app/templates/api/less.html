.pixel-pattern

Generate styles of pixel figure. For selector in scope it set width and height big enough to contain your figure. For nested selector .pixel-pattern-pixel it set multiple box shadows for actual figure.

.pixel-pattern( @pixel-list );
@pixel-list
List of pixels separated by comma. Single pixel is set of x, y coords and color separated by space.

Note: you need to specify at least 2 pixels

Example:

//Less
.my-pixel-pattern {
    .pixel-pattern( 0 1 red, 1 0 blue );
}

and corresponding html

<div class="my-pixel-pattern">
    <div class="pixel-pattern-pixel">
</div>

.pixel-pattern-size

Use this mixin to scale your figure created with .pixel-pattern mixin. Size determine width and height of single pixel. 

.pixel-pattern-size( @size );
@size
Size of single pixel in any unit supported by browser (default 1em)

Example:

//Less
.my-pixel-pattern {
    .pixel-pattern-size(32px);
    .pixel-pattern( some pixels here ... );
}

Less part of Pixel Pattern Generator is collection of mixins which let you create and maintain pixel based figures.

This is core of whole library, since CSS is primary method of rendering.
